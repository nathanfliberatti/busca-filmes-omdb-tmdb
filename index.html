Projeto: Busca de Filmes (API IMDb via OMDb e API TMDb)

Este exemplo mostra como conectar uma página HTML com duas APIs reais da internet. Vamos buscar filmes pelo título usando a API gratuita da OMDb, que traz dados do IMDb, e a API do TMDb, que fornece informações adicionais sobre filmes, nesse caso, a busca pelo nome do ator.
